Formule Importante

Teorema lui Pitagora

a² + b² = c²

Într-un triunghi dreptunghic, pătratul ipotenuzei este egal cu suma pătratelor catetelor.

Formula de rezolvare a ecuației de gradul II

x = (-b ± √(b² - 4ac)) / 2a

Pentru ecuația ax² + bx + c = 0, unde a ≠ 0

Aria cercului

A = πr²

Unde r este raza cercului și π ≈ 3.14159

Derivata funcției de putere

(xⁿ)' = nxⁿ⁻¹

Derivata funcției f(x) = xⁿ, unde n este un număr real
